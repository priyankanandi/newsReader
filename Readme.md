# News Reader
## Setup

Install nodejs `12.x.x` 

```sh
brew install nodejs
```

Install dependencies

```sh
npm install
```

Start the server

```sh
npm start
```

To run tests

```sh
npm run test
```

With coverage 

```sh
npm run test -- --coverage
````

Visit `localhost:8080`

## VRT Setup

To run vrt tests

```sh
backstop test
````

To approve vrt tests

```sh
backstop approve
````
